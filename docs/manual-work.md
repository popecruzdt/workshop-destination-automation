# Manual Workshop Preparation Work

This document tracks manual setup tasks that must be completed before or during workshop deployment. These steps cannot be automated through Ansible/AAP and require human intervention or external access.

## Required Manual Tasks

- [ ] Create aap-service-account user
- [ ] Clone workshop repo
- [ ] Download AAP install tarball
- [ ] Invite users to Dynatrace tenant
- [ ] Identify and locate workshop instance credential and private key
- [ ] Generate Dynatrace platform token

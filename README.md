# workshop-destination-automation

Workshop lab demonstrating **Dynatrace** observability + **Red Hat Ansible Automation Platform** automation on a containerized **AI workload** running with **Podman**.

## What's in this repo?

| Directory | Description |
|---|---|
| `app/` | AI Travel Advisor — FastAPI application source code and `Containerfile` |
| `ansible/` | Ansible project — playbooks and roles for build, deploy, and monitoring setup |
| `dynatrace/` | Dynatrace dashboard and configuration resources |
| `docs/` | Workshop guide with step-by-step instructions |
